## 💰 Finance Data Analysis Project

## 📌 Overview

This project focuses on analyzing a financial transactions dataset using Python.
It includes data cleaning, transformation, and visualization to extract meaningful insights from financial data.


## 🎯 Objectives

* Clean and preprocess raw financial data
* Handle missing values and duplicates
* Perform data analysis and summarization
* Visualize financial trends and patterns

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## 📂 Dataset Features

The dataset contains:

* Transaction Amount
* Net Amount
* Tax Amount
* Currency
* Transaction Date
* Company
* Department
* Analyst
* Approval Status
* Sector


## 🔧 Data Cleaning Steps

* Removed missing values in important columns
* Converted date column to proper datetime format
* Filled missing categorical values with "Unknown"
* Removed duplicate records


## 📊 Analysis Performed

### 1. Transaction Type Analysis

* Total amount grouped by transaction type

### 2. Sector Analysis

* Comparison of Net Amount and Tax Amount by sector

### 3. Monthly Trends

* Total transaction amount over months

### 4. Approval Status

* Distribution of approved vs rejected transactions

### 5. Top Analysts

* Top 5 analysts based on total transaction amount

### 6. Department Analysis

* Average transaction amount per department


## 📈 Visualizations

The project includes:

* Bar charts
* Line graphs
* Pie charts
* Horizontal bar charts


## 📌 Key Insights

* Identified highest transaction types
* Observed monthly financial trends
* Analyzed approval patterns
* Highlighted top-performing analysts

   ```bash

   ```

   ```bash

   ```

3. Run all cells step by step
